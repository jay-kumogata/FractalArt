## How to Run

Please execute the following on the Pyxel (version 2.0.13) environment.

	> python julia.py
  
## How to Control

- [up-arrow]: Move up 
- [down-arrow]: Move down
- [right-arrow]: Move right
- [left-arrow]: Move left
- [Z]: Zoom in
- [X]: Zoom out
